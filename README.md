
# PP02 — Производственная практика (Системное администрирование)

## Описание

Данный репозиторий содержит материалы производственной практики ПП.02 по направлению «Системное администрирование». В ходе практики выполнялись задачи по администрированию информационной системы нефтегазодобывающего предприятия, автоматизации работы системного администратора и обеспечению стабильной и безопасной работы компьютерной инфраструктуры промысловых и офисных подразделений.

---

## Содержание репозитория

### `docs/`
Документация по практике:
- Отчёт по производственной практике
- Описание предметной области (Нефтегазовая промышленность)
- Техническое задание на разработку системы управления промыслом
- UML-диаграммы (Use Case, Class, Sequence, Activity, State, Component)

---

## Цель работы

Целью производственной практики являлось получение практических навыков системного администрирования в условиях нефтегазового предприятия, автоматизация задач обслуживания ИТ-инфраструктуры, обеспечение сохранности производственных данных и повышение надежности работы информационных систем, поддерживающих процессы добычи и учёта углеводородов.

## Результаты

В ходе практики:
- изучены задачи системного администратора нефтегазодобывающего управления
- разработаны специализированные скрипты автоматизации с учётом отраслевой специфики
- реализовано резервное копирование баз данных по скважинам и мониторинг серверного оборудования
- выполнено моделирование системы управления промыслом с помощью UML-диаграмм
- оформлен отчёт по практике с описанием инфраструктуры предприятия
- создан публичный репозиторий с результатами работы
